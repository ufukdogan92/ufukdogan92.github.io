Mysql, Posgresql, Mssql gibi veritabanlarını kullanabiliyorum (arayüz veya komut satırında). Son zamanlarda Symfony ve Django ile çalışmaya başladım. Django ve Django Rest Framework ile uygulamalar yapıyorum. Yazdığım Web Servisleri PHP veya Angular 1.* ile kullanabiliyorum. Projeleri Github veya Bitbucket üzerinden geliştiriyorum. Sunucu olarak nginx veya apache kullanırum. İşletim sistemi olarak Linux dağıtımlarını tercih ediyorum. Bu özelliklerime ilave olarak React, Redis, Angular4 uygulamaları yaparak kendimi daha çok geliştirmek istiyorum.
